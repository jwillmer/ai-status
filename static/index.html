      AI Status

      A live dashboard for AI agent sessions. Each session is a .md file on disk; this app renders it in real time while your agents write to it.

      Project page: github.com/jwillmer/ai-status — source, releases, and issues.

      How it works

      1. Click ＋ to create a session. The app gives you a file path.
      2. Click the path to copy it. Paste it into Claude (or any agent).
      3. The agent writes status updates to that file.
      4. This page re-renders live on every save.

      Companion skill

      Pair this with the status-orchestrator skill. It makes Claude act as orchestrator/gatekeeper: maintains the status file, delegates work to background subagents, holds them to an industry-standard quality bar.

      ⬇ Download skill  or view the source on GitHub.

      In Claude: Customize → Skills → Install skill, select the downloaded .skill file.

      Tips

      • Double-click a session title to rename.
      • Archive old sessions to keep the sidebar clean.
      • Pin (☆ on hover) keeps important sessions on top.
      • 🔕/🔔 bell toggles desktop notifications for file changes across all sessions while the tab is hidden.
      • The app lives in the system tray — closing the tab leaves the server running. Quit only from the tray.
      • Ctrl+N = new session.
